# 组织建设论 · 操作指南

> "政策和策略是党的生命，各级领导同志务必充分注意，万万不可粗心大意。" ——毛泽东

## 整风运动方法论

### 核心思想
组织需要定期"整风"——不是清洗，是自我改进。通过系统性地反对错误倾向来提升组织战斗力。

> "惩前毖后，治病救人。" ——整风的根本方针

### 三种需要反对的错误倾向

| 错误倾向 | 原始定义 | 商业组织中的表现 |
|---------|---------|----------------|
| **主观主义** | 不从实际出发，凭主观想象做决策 | PM不调研就写需求、管理层闭门造车、用直觉代替数据 |
| **宗派主义** | 小圈子、山头主义、排斥异己 | 部门墙、"老人帮"排斥新人、只提拔自己人 |
| **党八股** | 形式主义、空话废话、文件治国 | 无实质内容的会议、PPT汇报文化、流程大于结果 |

> "无实事求是之意，有哗众取宠之心。华而不实，脆而不坚。自以为是，老子天下第一，'钦差大臣'满天飞。" ——《整顿党的作风》

毛泽东送给这类人一副对子：

> **"墙上芦苇，头重脚轻根底浅；山间竹笋，嘴尖皮厚腹中空。"**

→ 商业翻译：那些只会做PPT、不去一线调研、说起来头头是道但从不下手做的人，就是这副画像。

### 组织"整风"操作步骤

**1. 学习阶段（认识问题）**
- 让团队成员认识到什么是主观主义/宗派主义/形式主义
- 不是抽象地讲，而是用具体案例——最好是本团队发生过的

**2. 对照检查（发现问题）**
- 每个人对照检查自己：我有没有犯过这些错误？
- 不是相互揭发，是自我反省为主

**3. 整改阶段（解决问题）**
- 找到具体的改进措施
- 建立防止问题复发的机制

## 批评与自我批评

> "对以前的错误一定要揭发，不讲情面，要以科学的态度来分析批判过去的坏东西，以便使后来的工作慎重些，做得好些。这就是'惩前毖后'的意思。但是我们揭发错误、批判缺点的目的，好像医生治病一样，完全是为了救人，而不是为了把人整死。"

### 批评与自我批评的操作规范

**自我批评先行**：
- 领导先做自我批评，不是让下属先暴露
- 自我批评要具体——"上周我在X决策上犯了Y错误，原因是Z"
- 不是形式化的"我有不足"——那是党八股

**批评的原则**：
- **对事不对人** — "这个方案的问题是..."，不是"你这个人的问题是..."
- **有据可查** — 用事实和数据，不凭印象
- **建设性** — 批评完要给改进建议
- **有限度** — 批评是为了解决问题，不是为了打击人
- **有回应** — 被批评者要回应，不是单方面审判

### 复盘会模板（基于批评与自我批评）

```
项目/决策复盘：[名称]

一、事实回顾
- 目标是什么？
- 实际结果如何？
- 时间线上发生了什么？

二、自我批评
- 我在哪些环节判断有误？
- 我忽略了什么信息？
- 我的行为方式有什么问题？

三、改进措施
- 如果重来，我会怎么做不同？
- 需要建立什么机制防止复发？
- 需要什么资源或支持？

原则：惩前毖后，治病救人。目的是改进，不是追责。
```

## 民主集中制

> "在民主基础上的集中，在集中指导下的民主。"

### 核心机制

**民主**：决策前，充分听取意见、讨论方案、收集反对声音
**集中**：决策时，基于充分讨论做出统一决定，所有人执行

### 商业应用

**决策流程设计**：

| 阶段 | 规则 | 说明 |
|------|------|------|
| 讨论期 | 完全开放 | 所有人可以自由表达意见，包括反对意见 |
| 决策期 | 负责人拍板 | 基于充分讨论，由决策者做出最终决定 |
| 执行期 | 统一执行 | 无论之前是否同意，决策做出后统一执行 |
| 回顾期 | 开放检验 | 执行后回顾决策效果，可以修正 |

**两个常见错误**：
1. **只有民主没有集中** → 讨论来讨论去没有结论，效率极低
2. **只有集中没有民主** → 领导一言堂，决策质量差，团队执行不积极

## 论十大关系（平衡术）

> "提出这十个问题，都是围绕着一个基本方针，就是要把国内外一切积极因素调动起来，为社会主义事业服务。" ——《论十大关系》(1956)

### 企业管理中的十大关系

受《论十大关系》启发，企业管理中需要平衡的关键对立关系：

| 关系 | 失衡的表现 | 平衡的做法 |
|------|-----------|-----------|
| **研发 vs 销售** | 只做产品不卖、或只卖不做产品 | 产品驱动增长，销售检验产品 |
| **短期 vs 长期** | 只追求季度目标，或只画饼不落地 | 70%资源做短期，30%投长期 |
| **速度 vs 质量** | 粗制滥造，或过度完美主义 | 对外功能求快，对内基础设施求质 |
| **自建 vs 外购** | 什么都自己造，或过度依赖第三方 | 核心能力自建，非核心可外购 |
| **集权 vs 分权** | 什么都审批，或完全放任 | 战略集中，执行下放 |
| **现有业务 vs 新业务** | 只守不攻，或盲目多元化 | 现金牛养新业务，新业务不能耗死现金牛 |
| **大客户 vs 小客户** | 过度依赖一个大客户 | 大客户求深度，小客户求覆盖 |
| **内部 vs 外部** | 组织内耗严重，或只看外部忽视内建 | 内部效率是对外竞争的基础 |
| **标准化 vs 定制化** | 产品太标准不灵活，或太定制不可规模化 | 平台标准化，上层可定制 |
| **人才 vs 成本** | 不愿意付高薪，或过度烧钱 | 核心岗位不省钱，非核心岗位控成本 |

### 使用方法

当面对企业管理决策时，用这张表做"体检"：
1. 当前哪组关系失衡了？
2. 失衡的原因是什么？
3. 如何调整到动态平衡？

## 干部选拔原则

> "政治路线确定之后，干部就是决定的因素。"

### 选人四标准

| 标准 | 说明 | 检验方法 |
|------|------|---------|
| **德** | 价值观与团队一致 | 在压力下的选择，在利益面前的态度 |
| **能** | 做事的能力 | 过往业绩，解决复杂问题的实际案例 |
| **勤** | 投入度和执行力 | 面对困难时的韧性和行动力 |
| **绩** | 可量化的成果 | 带来了什么实际结果 |

### 反面标准

不用什么人：
- **只会说不会做的** → "马克思主义从前门进来，从后门出去"
- **只看上不看下的** → 对上级毕恭毕敬，对下属颐指气使
- **搞小圈子的** → 宗派主义，破坏团结
- **不从实际出发的** → 用通用方法硬套，不看具体情况
